# mini-chatbot

This is a mini chatbot built in Python using tkinter.  
I wrote and tested the entire project on my phone using Pydroid 3.

## Features
• Simple text based chatbot  
• GUI built with tkinter  
• Basic rule based responses  
• Lightweight and easy to run  

## Requirements
• Python 3  
• tkinter (included by default)

## How to Run
```bash
python chatbot.py
